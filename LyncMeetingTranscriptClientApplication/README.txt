Install instructions:
================================================
1. The following table describes the three registry entries. Add the context application GUID ({97AD7B8A-3220-4855-8D1E-E70BB0973C4D}) as a key under either of these two paths:
HKEY_CURRENT_USER\Software\Microsoft\Communicator\ContextPackages
HKEY_LOCAL_MACHINE\Software\Policies\Microsoft\Communicator\ContextPackages
